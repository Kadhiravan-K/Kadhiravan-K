# Method A simple to create with tools.

 https://profile-readme-generator.com/

# Method B - manual setup with more custimization.
> A simple step-by-step guide to create a professional GitHub Profile README.


# 🥇 Step 1 - Create Your Profile Repository

1. Login to GitHub.
2. Click **New Repository**.
3. Repository name **must be exactly your GitHub username**.

Example

```
Username:
aadhi

Repository:
aadhi 
```

4. Select **Public**
5. Check **Add a README file**
6. Click **Create Repository**

🎉 Your GitHub Profile README is now created.

---

# ✏️ Step 2 - Edit README.md

Open your repository.

Click

```
README.md
    ↓
✏ Edit
```

Now you can start editing your profile.

Click **Commit Changes** after every update.

# 📝 Step 3 - Learn Basic Markdown

| Markdown | Output |
|----------|--------|
| `# Title` | Heading |
| `## Title` | Sub Heading |
| `**Bold**` | **Bold** |
| `*Italic*` | *Italic* |
| `- Item` | Bullet List |
| `[Google](https://google.com)` | Link |
| `![Image](url)` | Image |

📖 Learn More

https://www.markdownguide.org/

# ⚙️ Step 16 - GitHub Actions

GitHub Actions automatically update:

- 🐍 Snake Animation
- 📊 Metrics
- ⌚ WakaTime

- 📰 Blog Posts

